# Simple repo to demo faunadb-js modularization

### NodeJS CJS
```
node NodeJS_cjs.js
```

### NodeJS ESM

```
node --es-module-specifier-resolution=node NodeJS_esm.mjs
```


#### React with ESM

```
cd ./react-faunadb

yarn start
```
